Dear Pocket Guide Fans,

I'm sure it's obvious now that I've stopped playing FFXIV. While I've been following the community on-and-off since then, I don't think I'm going to make it back into the game. This is a result of heavy burn out and a lack of time to invest in generating content for the site.

That being said, the Pocket Guide will continue to stay online, and I'm happy to continue updating it with bug fixes and interface improvements if they're suggested to me.

This has been an incredibly fun project for me, and I'd like to see it continued.

If any of you would like to see the site updated with new content and are willing to format guides yourselves, I'm more than happy to explain the template and process. Once you've submitted the content to me I'd be happy to publish it here.

Feel free to reach out if this interests you!

Cheers, Josh
